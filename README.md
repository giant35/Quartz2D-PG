# 介绍

Core Graphics, 也就是众所周知的 Quartz 2D，是一种高级的2维绘画引擎，能用于iOS、tvOS 及 macOS 应用程序开发。Quartz 2D 提供轻量级低级与设备及分辨率无关的2D渲染。

Quartz 2D API 易于使用且提供丰富的功能，如透明层、路径绘制、离屏渲染、高级颜色管理、反锯齿渲染及PDF文档创建解析显示。



## 面向的读者

给需要开发下列功能的开发者：

* 绘图
* 图形编辑应用
* 创建或显示图片
* 操作 PDF
